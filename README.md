# Crowded Scene

Crowded Scene is a small 2D game prototype built with Godot.  
The player walks through a busy street delivering assignments to different stores while avoiding pedestrians.

This project is part of the Ubisoft Mentorship program submission.

## Current Features

- Player movement in a street environment
- NPC crowd system with vertical pedestrian flow
- Store interaction system with assignments
- Basic collision system between player and NPCs
- Tutorial and game scene

## How to Run

### Option 1 – Play in Browser
Use the browser build: https://schardot.itch.io/release-090

### Option 2 – Run Locally
1. Install **Godot 4**
2. Clone the repository
3. Open the project in Godot
4. Run `game.tscn`

## Current Limitations

- NPCs are only spawned once at the start
- NPC collisions can cause characters to get stuck
- The game ends after completing 5 assignments
